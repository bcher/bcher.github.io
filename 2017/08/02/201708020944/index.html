比特币分裂！独立而出的“比特币现金”正式运营

发表于 2017-08-02 |

本周二，全球最大，也是最有名的数码货币“比特币”正式分裂。由一小群比特币“矿工”和投资方所推出的“比特币现金”（Bitcoin Cash）开始挂牌运营。

此次分裂的主要原因是该人群对比特币框架中交易速度的不满。在现有的框架下，比特币的每秒交易量理论上最高只有7笔（实际为3、4笔），相比之下，国际信用卡Visa、Master Card以及中国银联的峰值则在每秒5-6万笔的范围，而支付宝在“双11”的峰值甚至已达每秒8.6万笔。随着比特币扩散的速度越来越快，接受率越来越高，这种程度的交易速度已经开始跟不上了。

由于比特币是基于“分布式账本”，即区块链的货币，其交易速度是由区块链中每个“区块”的大小所决定的。为了解决这个问题，比特币现金将其“区块”大小定为8MB，是比特币1MB“区块”的8倍。理论上，这可以让比特币现金的秒交易为比特币的八倍，达到每秒56笔。

不过，除了“区块”大小的区别，比特币现金的框架与比特币基本相同。比特币现金甚至还保留了比特币从始至今的所有交易在其区块链上。（如果你所使用的比特币交易所支持比特币现金，你将会发现你多出来一个比特币现金的账户，里面有与你所持比特币数量相同的比特币现金。）

其实比特币早就有分裂的迹象了。早在两年前，就有开发者表示比特币区块链的结构将会成为它的瓶颈，应该尽早扩大区块的大小，提高交易速度。

不过，负责开发维护比特币的个人和机构一直在反对这种声音。他们认为，如果提高区块的大小，小型“矿工”将无法承担起那些能“挖”（计算）动大区块所需设备的成本，导致他们被踢出全球区块链。而一旦区块链是由大型矿工组织承担的，这很有可能导致比特币受到一个或多个大鳄的控制，从基础上动摇其对等模式（P2P）。

所以，此次比特币现金的独立只不过是多年斗争的结果。在分裂后，全球绝大部分的矿工和开发者都将继续支持比特币，比特币现金则将依靠ViaBTC的挖矿池运作。ViaBTC代表了全球6%的比特币计算能力。不过，若是比特币现金的价格能够重现比特币的神话，想必会有更多的矿工选择加入。
